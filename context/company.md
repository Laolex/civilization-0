# Company Context

## One-Liner
Civilization-0 is a verifiable provenance layer for agentic AI — it proves *why* an autonomous agent decided something, trustlessly, on 0G.

## Elevator Pitch
As AI agents get more autonomous, "trust our logs" stops being good enough — especially where decisions are adversarial, on-chain, multi-party, or regulated. Civilization-0 reasons each decision on 0G Compute (TEE-verified) and archives its full causal chain — Memory → Belief → Decision → Event → Trace — to 0G Storage, where *anyone* can independently verify it with no trust in the operator and no private key. We prove it with the hardest possible demo: a persistent AI society that runs itself, verifiably. The civilization is the proof; the verifiable provenance layer is the product.

## Key Differentiators
1. **Keyless, trustless verification** — anyone recovers a decision record from 0G Storage by root hash alone, no signer, no trust in the operator. "Rip out 0G and the product doesn't exist."
2. **`drivers`, not just logs** — records the brain-weighted subset that the decision *actually hinged on* (Memory → Belief weights), not just everything that was retrieved. That's the provenance moat.
3. **Reasoning itself is verified** — decisions run on 0G Compute with cryptographic `verified: true` (TEE-verified inference), not just stored after the fact.
4. **One API call** — `civ.trace(...)` wraps 0G Compute + 0G Storage and returns a public `verifyUrl`. Drops into an existing agent stack.
